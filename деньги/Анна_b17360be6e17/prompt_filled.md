## Что сделать
Напиши совет на ближайшее время строго про деньги и реализацию. Опирайся на
текущий период Вимшоттари (махадаша / антардаша / пратьянтардаша) и на то, какие
планеты периода активируют финансовые коды клиента.

Первый вариант совета собирается программно движком `advice_generator.py`
(`build_advice_context(csv, 'money', razbor_path=...)` → `draft_advice`). Этот
промпт — ориентир для проверки и ручной доработки Кайей.

## Структура (как в движке)
- Вступление: имя + headline из разбора + текущий период даши (одно предложение).
- По каждой планете периода: период (махадаша/антардаша/пратьянтардаша) + планета
  и дом (только номер) + что несёт дом в теме денег (одно предложение) + ДНК
  (если соединение) или знак+накшатра (если соединения нет) + один денежный шаг.
- Смена периода в ближайшие 3 месяца: что за планета включается, дом, знак,
  накшатра, качество, ДНК (если есть).
- Годовой прогноз: уникальный абзац под карту (планеты, дома, ДНК, развилка
  периода по деньгам). ЗАПРЕЩЕНО: «что вообще будет», шаблонные «когда усиливать
  шаги». URL: «Подробнее по ссылке: …»

## Требования к тексту
- Обращение на «вы», тёплый деловой тон, без эзотерического жаргона.
- Только про деньги: доход, продажи, клиенты, продукт, система, инвестиции.
- ДНК = ТОЛЬКО соединение. Аспект — это помощь / возможность / незавершённое /
  долг из прошлых жизней, но НЕ ДНК.
- Действия — из словаря `NAKSHATRA_ACTIONS_MONEY`. Запрещено «починить,
  приготовить» и прочие бытовые/любовные действия.
- Качество периода — из `NAKSHATRA_QUALITIES_MONEY` (через призму заработка).
- Без гарантий, без дат-пророчеств. Длинные тире не используем.

## Формат вывода
Только текст совета. Кайя проверяет, правит и сохраняет в
`advice_after_review.md`, после чего программа публикует его в кабинет.
